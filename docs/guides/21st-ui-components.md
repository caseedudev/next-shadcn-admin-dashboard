### MARKETING BLOCKS 상세 설명

* **Announcements (공지)**
    * **설명:** 신제품 출시, 이벤트, 점검 등 중요한 소식을 알리는 바 형태의 컴포넌트입니다.
    * **사용처:** 홈페이지 최상단 (GNB 위쪽).
    * **예시:** "지금 가입하면 50% 할인 쿠폰 증정!" 알림 바.

* **Backgrounds (배경)**
    * **설명:** 섹션의 뒤쪽에 위치하여 분위기를 결정하는 요소입니다. 단색, 그라데이션, 패턴 등이 포함됩니다.
    * **사용처:** 페이지 전체 혹은 특정 섹션의 시각적 구분.
    * **예시:** 추상적인 그래픽이나 부드러운 파스텔톤 배경.

* **Borders (테두리/구분선)**
    * **설명:** 콘텐츠 블록 간의 경계를 나누거나 강조할 때 사용하는 디자인 요소입니다.
    * **사용처:** 섹션 사이의 구분, 카드 뉴스 형태의 외곽선.
    * **예시:** 점선이나 굵은 실선으로 영역을 확실히 나누는 디자인.

* **Calls to Action (CTA/구매 유도)**
    * **설명:** 사용자의 클릭이나 특정 행동(가입, 구매)을 유도하는 버튼과 문구 세트입니다.
    * **사용처:** 랜딩 페이지 중간 및 하단.
    * **예시:** "30일 무료 체험 시작하기" 버튼이 강조된 섹션.

* **Clients (고객사)**
    * **설명:** 해당 서비스를 사용하는 유명 기업들의 로고를 모아 보여주는 섹션입니다.
    * **사용처:** 신뢰도를 높여야 하는 기업용(B2B) 웹사이트.
    * **예시:** "1,000개 이상의 기업이 함께합니다" 문구와 로고 모음.

* **Comparisons (비교)**
    * **설명:** 자사 서비스의 요금제 차이나 경쟁사와의 기능을 한눈에 비교하는 표입니다.
    * **사용처:** 가격 정책 페이지, 제품 상세 설명.
    * **예시:** 무료 vs 프로 버전의 기능 차이 비교표.

* **Docks (독/고정 바)**
    * **설명:** 화면 하단 등에 고정되어 주요 메뉴로 바로 이동하게 돕는 플로팅 바입니다.
    * **사용처:** 모바일 웹이나 앱의 하단 네비게이션.
    * **예시:** 화면 아래에 항상 떠 있는 홈, 검색, 마이페이지 버튼.

* **Features (기능 소개)**
    * **설명:** 제품이 가진 주요 장점이나 기능을 아이콘과 함께 설명하는 블록입니다.
    * **사용처:** 랜딩 페이지 메인 본문.
    * **예시:** "빠른 속도", "보안성", "쉬운 사용법" 등을 나열한 섹션.

* **Footers (푸터)**
    * **설명:** 웹사이트의 최하단 영역으로 기업 정보, 약관, SNS 링크 등이 담깁니다.
    * **사용처:** 모든 페이지의 가장 아랫부분.
    * **예시:** 주소, 사업자 번호, 고객센터 번호가 적힌 하단 영역.

* **Heroes (히어로 섹션)**
    * **설명:** 웹사이트 접속 시 가장 먼저 보이는 거대한 상단 영역입니다.
    * **사용처:** 홈페이지의 첫 인상을 결정하는 메인 상단.
    * **예시:** 배경 이미지 위에 큰 메인 문구와 가입 버튼이 있는 화면.

* **Hooks (훅/시선 강탈)**
    * **설명:** 사용자의 호기심을 자극하거나 강력한 문구로 이목을 끄는 짧은 문구 블록입니다.
    * **사용처:** 페이지 전환 구간이나 스크롤 중간.
    * **예시:** "단 3초 만에 당신의 업무 효율을 바꿔보세요."

* **Images (이미지)**
    * **설명:** 제품 사진, 그래픽 디자인 등 시각적 자료를 배치하는 레이아웃입니다.
    * **사용처:** 설명이 필요한 모든 구간.
    * **예시:** 제품 실제 사용 모습을 담은 고화질 사진 슬라이드.

* **Maps (지도)**
    * **설명:** 오프라인 매장이나 사무실의 위치를 보여주는 대화형 지도 블록입니다.
    * **사용처:** '오시는 길', '매장 찾기' 페이지.
    * **예시:** 구글 지도나 네이버 지도가 삽입된 회사 소개 섹션.

* **Navigation Menus (네비게이션 메뉴)**
    * **설명:** 사용자가 원하는 페이지로 이동할 수 있게 돕는 상단 메뉴 바입니다.
    * **사용처:** 웹사이트 최상단 (GNB).
    * **예시:** 로고 옆에 나열된 '소개', '가격', '커뮤니티' 메뉴.

* **Pricing Sections (가격 섹션)**
    * **설명:** 유료 서비스의 구독료나 구매 가격을 카드 형태로 보여주는 블록입니다.
    * **사용처:** 요금 안내 페이지.
    * **예시:** 월간/연간 구독료가 적힌 3개의 가격 카드.

* **Scroll Areas (스크롤 영역)**
    * **설명:** 가로나 세로로 스크롤하여 더 많은 정보를 볼 수 있게 만든 영역입니다.
    * **사용처:** 많은 양의 이미지나 카드를 좁은 공간에 보여줄 때.
    * **예시:** 가로로 넘겨보는 최신 뉴스레터 목록.

* **Shaders (쉐이더/효과)**
    * **설명:** 배경이나 이미지 위에 덧씌우는 그림자, 빛, 질감 등의 특수 효과입니다.
    * **사용처:** 디자인의 깊이감이나 고급스러움을 더할 때.
    * **예시:** 이미지 하단을 어둡게 처리하여 위에 쓴 글자가 잘 보이게 하는 효과.

* **Testimonials (후기)**
    * **설명:** 실제 사용자들의 추천사나 별점 리뷰를 모아 놓은 섹션입니다.
    * **사용처:** 구매 결정 직전의 신뢰 구축 단계.
    * **예시:** "이 서비스를 쓰고 매출이 2배 늘었어요!"라는 고객 인터뷰.

* **Texts (텍스트 블록)**
    * **설명:** 긴 본문, 제목, 설명 등 글자로만 이루어진 다양한 서식의 블록입니다.
    * **사용처:** 블로그 포스트, 상세 가이드라인.
    * **예시:** 이용 약관이나 서비스 이용 방법 안내글.

* **Videos (비디오)**
    * **설명:** 홍보 영상이나 튜토리얼 영상을 재생할 수 있는 블록입니다.
    * **사용처:** 제품 시연, 브랜드 필름 소개.
    * **예시:** 유튜브 영상을 임베드(삽입)한 제품 소개 섹션.

### UI COMPONENTS 상세 설명

* **Accordions (아코디언)**
    * **설명:** 클릭하면 내용이 아래로 펼쳐지고 다시 클릭하면 접히는 컴포넌트입니다.
    * **사용처:** FAQ(자주 묻는 질문), 좁은 공간에 많은 정보를 담아야 할 때.
    * **예시:** 질문 제목을 누르면 아래로 답변이 나타나는 목록.

* **AI Chats (AI 채팅)**
    * **설명:** 사용자와 인공지능이 대화를 주고받을 수 있는 채팅창 인터페이스입니다.
    * **사용처:** 고객 센터, AI 비서 서비스.
    * **예시:** 화면 우측 하단에 떠 있는 챗봇 상담창.

* **Alerts (알림 창)**
    * **설명:** 사용자에게 중요한 정보나 경고, 성공 메시지를 전달하는 피드백 박스입니다.
    * **사용처:** 설정 변경 후 확인, 시스템 오류 발생 시.
    * **예시:** 상단에 나타나는 "성공적으로 저장되었습니다"라는 파란색 박스.

* **Avatars (아바타)**
    * **설명:** 사용자의 프로필 사진이나 이니셜을 원형 또는 사각형으로 보여주는 이미지 요소입니다.
    * **사용처:** 댓글창, 마이페이지, 팀원 목록.
    * **예시:** 채팅창 옆에 작게 표시된 사용자의 얼굴 사진.

* **Badges (배지)**
    * **설명:** 특정 상태나 숫자를 나타내기 위해 요소 옆에 붙는 작은 태그입니다.
    * **사용처:** 읽지 않은 알림 수 표시, 제품의 상태 표시.
    * **예시:** 종 모양 아이콘 위에 빨간색 숫자로 표시된 '3'.

* **Buttons (버튼)**
    * **설명:** 사용자가 클릭하여 특정 동작을 실행하게 하는 가장 기본적인 요소입니다.
    * **사용처:** 모든 상호작용이 필요한 곳.
    * **예시:** "저장하기", "취소", "더 보기".

* **Calendars (캘린더)**
    * **설명:** 날짜를 확인하거나 특정 날짜의 일정을 관리할 수 있는 달력 형태의 컴포넌트입니다.
    * **사용처:** 예약 시스템, 스케줄러.
    * **예시:** 숙박 예약 사이트의 날짜 선택 창.

* **Cards (카드)**
    * **설명:** 이미지, 제목, 본문을 하나의 박스 안에 담아 정보를 그룹화한 레이아웃입니다.
    * **사용처:** 콘텐츠 목록, 대시보드 요약 정보.
    * **예시:** 쇼핑몰의 상품 리스트 한 칸(사진+상품명+가격).

* **Carousels (캐러셀)**
    * **설명:** 여러 개의 배너나 이미지를 좌우 슬라이드로 넘겨볼 수 있는 컴포넌트입니다.
    * **사용처:** 메인 배너, 추천 상품 목록.
    * **예시:** 홈페이지 상단에서 자동으로 넘어가는 홍보 이미지들.

* **Checkboxes (체크박스)**
    * **설명:** 여러 항목 중 하나 이상을 중복 선택할 때 사용하는 선택 상자입니다.
    * **사용처:** 필터 설정, 약관 동의.
    * **예시:** "개인정보 수집 동의(필수)", "마케팅 수신 동의(선택)".

* **Date Pickers (날짜 선택기)**
    * **설명:** 날짜를 입력하거나 달력 팝업에서 날짜를 선택하게 돕는 입력창입니다.
    * **사용처:** 생년월일 입력, 여행 기간 설정.
    * **예시:** 출국 날짜를 선택하기 위해 클릭하면 나오는 달력 도구.

* **Dialogs / Modals (모달/팝업)**
    * **설명:** 현재 화면 위에 겹쳐서 나타나며, 사용자의 확인이나 입력을 요구하는 창입니다.
    * **사용처:** 회원가입 폼, 삭제 확인 알림.
    * **예시:** "정말 삭제하시겠습니까?"라고 묻는 중앙 팝업창.

* **Dropdowns (드롭다운)**
    * **설명:** 클릭하면 선택 가능한 목록이 아래로 펼쳐지는 컴포넌트입니다.
    * **사용처:** 언어 선택, 설정 옵션.
    * **예시:** '정렬 기준' 클릭 시 나오는 '최신순', '인기순' 목록.

* **Empty States (빈 화면)**
    * **설명:** 데이터나 검색 결과가 없을 때 보여주는 안내 화면입니다.
    * **사용처:** 장바구니가 비었을 때, 검색 결과 없음.
    * **예시:** "보관된 메시지가 없습니다"라는 문구와 함께 있는 아이콘.

* **File Trees (파일 트리)**
    * **설명:** 폴더와 파일의 계층 구조를 나무 모양으로 보여주는 목록입니다.
    * **사용처:** 관리자 페이지, 개발 도구, 문서 관리 시스템.
    * **예시:** 윈도우 탐색기의 왼쪽 폴더 구조 메뉴.

* **File Uploads (파일 업로드)**
    * **설명:** 사용자가 자신의 컴퓨터에 있는 파일을 서버로 올릴 수 있게 하는 영역입니다.
    * **사용처:** 프로필 사진 변경, 서류 제출.
    * **예시:** "이곳에 파일을 드래그하여 업로드하세요" 영역.

* **Forms (폼)**
    * **설명:** 이름, 이메일, 내용 등 여러 정보를 한꺼번에 입력받는 양식들의 집합입니다.
    * **사용처:** 가입, 문의하기, 주문서 입력.
    * **예시:** 이름, 연락처, 주소를 적고 마지막에 전송 버튼이 있는 섹션.

* **Icons (아이콘)**
    * **설명:** 텍스트 대신 의미를 전달하는 작은 시각적 기호입니다.
    * **사용처:** 메뉴, 버튼 내 의미 보강.
    * **예시:** 돋보기(검색), 톱니바퀴(설정), 집(홈).

* **Inputs (입력창)**
    * **설명:** 사용자가 텍스트를 직접 타이핑하여 입력할 수 있는 빈 칸입니다.
    * **사용처:** 검색어 입력, 아이디/비밀번호 입력.
    * **예시:** 로그인 화면의 'ID' 입력란.

* **Links (링크)**
    * **설명:** 클릭하면 다른 페이지나 외부 사이트로 연결되는 텍스트 요소입니다.
    * **사용처:** 본문 내 참조, 하단 약관 바로가기.
    * **예시:** 파란색 밑줄이 쳐진 "비밀번호를 잊으셨나요?".

* **Menus (메뉴)**
    * **설명:** 이동할 수 있는 경로를 모아놓은 목록입니다.
    * **사용처:** 상단바, 마우스 우클릭 시 나타나는 옵션.
    * **예시:** 파일, 수정, 보기 등이 나열된 메뉴 바.

* **Notifications (알림)**
    * **설명:** 실시간으로 발생하는 소식을 사용자에게 알리는 시스템 메시지입니다.
    * **사용처:** 메시지 도착, 업데이트 완료 안내.
    * **예시:** 화면 구석에 잠시 나타났다 사라지는 "새로운 댓글이 달렸습니다".

* **Numbers (숫자 입력)**
    * **설명:** 오직 숫자만 입력받거나, +, - 버튼으로 수량을 조절하는 컴포넌트입니다.
    * **사용처:** 상품 수량 선택, 나이 입력.
    * **예시:** 쇼핑몰 장바구니의 상품 개수 조절 버튼.

* **Paginations (페이지네이션)**
    * **설명:** 많은 양의 데이터를 여러 페이지로 나누어 보여주는 번호 매기기입니다.
    * **사용처:** 게시판 하단, 검색 결과 목록.
    * **예시:** 목록 아래의 [1] [2] [3] ... [다음] 번호.

* **Popovers (팝오버)**
    * **설명:** 특정 요소를 클릭하거나 마우스를 올릴 때 그 옆에 작은 설명 창이 뜨는 방식입니다.
    * **사용처:** 부가 설명 제공, 간이 프로필 보기.
    * **예시:** 아이디 위에 마우스를 올리면 뜨는 작은 사용자 정보 창.

* **Radio Groups (라디오 그룹)**
    * **설명:** 여러 선택지 중 단 하나만 선택할 수 있는 동그란 버튼 모음입니다.
    * **사용처:** 성별 선택, 결제 수단 중 하나 선택.
    * **예시:** 결제 방식 선택(카드 / 계좌이체 / 휴대폰).

* **Selects (선택 박스)**
    * **설명:** 드롭다운과 유사하며, 미리 정의된 옵션 리스트에서 값을 선택하는 창입니다.
    * **사용처:** 국가 선택, 생년 선택.
    * **예시:** 주소 입력 시 '시/도'를 고르는 선택창.

* **Sidebars (사이드바)**
    * **설명:** 화면의 왼쪽이나 오른쪽에 고정된 세로형 메뉴 영역입니다.
    * **사용처:** 대시보드 메뉴, 관리자 시스템 주 메뉴.
    * **예시:** 이메일 서비스 왼쪽의 '편지함', '보낸 편지함' 목록.

* **Sign Ins / Sign ups (로그인/회원가입)**
    * **설명:** 사용자 인증을 위한 전용 폼 페이지나 섹션입니다.
    * **사용처:** 서비스 접근 제어.
    * **예시:** 이메일과 비밀번호 입력란이 있는 로그인 페이지.

* **Sliders (슬라이더)**
    * **설명:** 조절 바를 좌우로 움직여서 수치를 변경하는 컴포넌트입니다.
    * **사용처:** 음량 조절, 가격 범위 설정.
    * **예시:** 쇼핑몰 필터에서 '0원 ~ 10만원' 범위를 조절하는 바.

* **Spinner Loaders (로딩 스피너)**
    * **설명:** 데이터가 처리 중임을 알려주는 회전하는 애니메이션 아이콘입니다.
    * **사용처:** 페이지 로딩 중, 데이터 업로드 중.
    * **예시:** 원형 아이콘이 뱅글뱅글 돌아가는 화면.

* **Tables (표)**
    * **설명:** 격자 형태로 데이터를 나열하여 비교하기 쉽게 만든 컴포넌트입니다.
    * **사용처:** 거래 내역, 학생 명단, 데이터 통계.
    * **예시:** 엑셀 시트처럼 정리된 주문 내역 리스트.

* **Tabs (탭)**
    * **설명:** 같은 영역에서 서로 다른 카테고리의 내용을 교체하며 보여주는 메뉴입니다.
    * **사용처:** 마이페이지(주문내역/리뷰관리/정보수정).
    * **예시:** 포털 사이트 메인의 '뉴스', '연예', '스포츠' 탭.

* **Tags (태그)**
    * **설명:** 키워드를 강조하거나 분류할 때 사용하는 작은 캡슐 형태의 요소입니다.
    * **사용처:** 관심사 표시, 게시물 카테고리.
    * **예시:** 게시물 하단의 #디자인, #개발 같은 해시태그.

* **Text Areas (텍스트 영역)**
    * **설명:** 여러 줄의 긴 글을 입력받을 수 있는 커다란 입력창입니다.
    * **사용처:** 문의 내용 작성, 댓글 작성.
    * **예시:** "문의 내용을 상세히 적어주세요"라는 큰 입력 박스.

* **Toasts (토스트)**
    * **설명:** 화면 하단에서 슬쩍 올라왔다가 자동으로 사라지는 짧은 알림 메시지입니다.
    * **사용처:** 링크 복사 완료, 전송 성공 알림.
    * **예시:** "복사되었습니다"라는 작은 검은색 팝업 메시지.

* **Toggles (토글/스위치)**
    * **설명:** 켜기/끄기(On/Off) 상태를 전환하는 스위치 버튼입니다.
    * **사용처:** 설정 페이지, 다크모드 전환.
    * **예시:** 스마트폰 설정의 'Wi-Fi 켜기/끄기' 스위치.

* **Tooltips (툴팁)**
    * **설명:** 마우스를 올렸을 때 해당 요소에 대한 짧은 설명을 보여주는 말풍선입니다.
    * **사용처:** 아이콘의 의미 설명, 용어 풀이.
    * **예시:** '?' 아이콘에 마우스를 대면 뜨는 "이 기능은...입니다" 문구.
